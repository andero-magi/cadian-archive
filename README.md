# Running
```
npm i
cd frontend
npm i
cd ..
cd express-project
npm i
cd ..
npm run run
```

To run both backend and frontend: 
```
npm run run
```

To run backend by itself:
```
cd express-project
npm run start
```

To run frontend by itself
```
cd frontend
npm run dev
```
